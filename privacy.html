개인정보 처리방침

이 방침은 테토/에겐 얼굴 분석기(이하 “서비스”)에서 사진과 접속 정보가 어떻게 처리되는지 설명합니다. 서비스는 회원가입 기능을 제공하지 않으며 이름, 전화번호 또는 주소를 직접 입력받지 않습니다.

1. 업로드 사진과 분석 결과

사용자가 선택한 사진은 브라우저에서 미리보기, 모델 분석, 결과 PNG 생성에 사용됩니다. 서비스 운영자가 사진을 수집해 보관하는 별도 계정이나 데이터베이스 기능은 없습니다. 페이지를 닫거나 사진을 다시 선택하면 브라우저 화면의 해당 상태는 사라집니다.

카카오톡 공유를 사용하면 공유 카드 이미지를 생성하기 위해 사진이 블러 처리된 결과 이미지가 Kakao 서비스로 업로드됩니다. 왓츠앱 공유는 결과 문구와 사이트 링크를 전달하며 사진 파일을 자동 첨부하지 않습니다. 결과 다운로드를 선택하면 원본 사진이 포함된 PNG가 사용자의 기기에 저장됩니다.

2. 자동으로 처리될 수 있는 정보

웹사이트 제공 과정에서 호스팅 사업자와 외부 리소스 제공자는 IP 주소, 브라우저 종류, 요청 시각, 방문 URL, 오류 기록과 같은 일반적인 서버 로그를 보안 및 서비스 제공 목적으로 처리할 수 있습니다. 서비스는 TensorFlow.js, Teachable Machine 및 Kakao JavaScript SDK 등 외부 리소스를 불러오므로 해당 제공자에게 네트워크 요청 정보가 전달될 수 있습니다.

3. 쿠키와 Google 광고

향후 Google AdSense가 적용되면 Google을 포함한 제3자 광고 사업자가 사용자의 이전 방문 기록을 바탕으로 광고를 제공하기 위해 쿠키, 웹 비콘, IP 주소 또는 기타 식별자를 사용할 수 있습니다. Google의 광고 쿠키를 통해 Google과 파트너는 이 사이트 또는 다른 사이트 방문 정보를 바탕으로 광고를 제공할 수 있습니다.

사용자는 Google 광고 설정에서 맞춤 광고를 관리하거나 사용 중지할 수 있습니다. Google이 파트너 사이트의 정보를 사용하는 방식은 Google 파트너 사이트 개인정보 안내에서 확인할 수 있습니다. 법령상 동의가 필요한 지역에는 광고 활성화 전에 Google이 요구하는 동의 관리 절차를 적용합니다.

4. 정보의 공유와 보관

서비스는 이용자의 사진이나 개인정보를 판매하지 않습니다. 법령상 의무가 있거나 보안 사고 대응에 필요한 경우를 제외하면 운영자가 직접 수집한 개인정보를 제3자에게 제공하지 않습니다. 외부 공유 버튼을 사용하면 이용자가 선택한 서비스의 약관과 개인정보 처리방침이 적용됩니다.

5. 이용자의 선택과 권리

사진 선택은 전적으로 자발적이며 언제든 페이지를 닫아 처리를 중단할 수 있습니다. 브라우저 설정에서 쿠키를 삭제하거나 차단할 수 있으나 일부 외부 기능이 제한될 수 있습니다. 개인정보 관련 질문이나 삭제 요청 대상이 있다고 판단되는 경우 문의 페이지를 이용해 주세요.

6. 아동 및 타인의 사진

서비스는 만 13세 미만 아동을 대상으로 설계되지 않았습니다. 보호자는 아동이 사진을 업로드하거나 공유하지 않도록 지도해야 합니다. 타인의 사진을 사용할 때는 반드시 적법한 권한과 당사자의 동의를 확보해야 합니다.

7. 방침 변경

서비스 기능, 광고 또는 관련 법령이 변경되면 이 방침을 수정하고 시행일을 이 페이지에 표시합니다.
